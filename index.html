Teams

  • Mail Routing

    Spam is only one of the challenges when working with millions of emails a day. Using a lot of Linux, open source technologies, and a variety of programming languages, we make sure that mail gets delivered on time!

  • Mail Storage

    No one ever, ever, EVER wants to lose an email, so we go crazy about large-scale storage and redundancy. Using Dovecot, Python, C, and Rackspace Cloud Files you'll have that mail when you need it!

  • Provisioning

    Managing over 350,000 Exchange mailboxes is no small task. We handle 10,000+ updates every day. With C# .Net, MSMQ, PowerShell, and REST APIs we've got the tools to keep it all running smoothly.

  • Email and Apps Control Panel

    Our Control Panel team puts the power at our customers' fingertips with a robust web application for service management. Using C# .NET, javascript, IIS, and SQL Server we make configuring email a breeze!

  • Automation and Tools

    “Appears we got here just in the nick of time. What does that make us?” Automation and data visualization are key components in managing a large-scale shiny email system. From load testing to large-scale deployments to customer trends, we’re creating the software that does it all for us! If you’re up for the challenges in making systems that manage themselves, we want you to help us! Technologies include Python, PHP, chef, MySQL, and OpenStack.

  • Cloud Servers and Images

    We are an Openstack project that provides key services such as registry, discovery, retrieval and management of virtual machine images. We strive to be highly available, extremely fault-tolerant and easily recoverable to be able to cater to the real-time needs of clouds. Combined together, we power the 2nd largest public cloud, Rackspace Cloud, in the industry and see ourself as a key contributor to Openstack. So, this is where all the action is, if 'Python', 'Agile Development', 'Open Source', 'Cloud' and 'Awesome' appeal to you!

  • Webmail

    Businesses need more than email, and that's where Webmail steps up with calendar, contacts, synchronization services, and a sophisticated web interface to rule them all. We keep productivity high with javascript, PHP, and MySQL!

  • Directory Synchronization

    Our customers are too busy running their business to manage mailboxes. We've got the tools to keep their email configuration in sync with their active directory. Using C# .Net and REST APIs we make configuring email seamless.

  • Cloud Control Panel

    Fanatical Support starts with a Control Panel that enables our customers to get things done faster and simpler than ever before. You'll work with Javascript, Ruby, and Python to turn our public REST APIs into a responsive and beautiful web interface that leads our customers to rave about their experience with our Cloud products.
